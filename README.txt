==== UPS File Linker ====

=== Usage ===
The program will automatically search for files matching the specified file extension in the set search folder.
These files will then be shown in the large treeview.
To link a file, select it and click the link button.

=== Configuration ===
The configuration window shows four main settings.
Crawl directory - set directory to search files in
Target directory - set directory to put target file in
File extension - set the file extension to search for
Target file - set the target file name

The configuration is saved in the file UPSLinker.txt under the local APPDATA folder.
Conf-File Location: User\AppData\Local\Alcasa\UPSLinker\pyIndexer.txt
